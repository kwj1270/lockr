# Lockr 앱 - 회원 프로필 관리 요구사항명세서

## 📋 1. 개요

### 1.1 목적

스포츠 동아리를 위한 **Lockr** 앱의 회원 프로필 정보 관리에 대한 요구사항을 정의합니다.

### 1.2 범위

- 개인 프로필 정보 관리
- 개인정보 공개 설정
- 회원 설정 및 탈퇴

---

## 👤 2. 회원 프로필 정보 구성

### 2.1 기본 프로필 정보

**필수 정보:**

- **실명** (본명)
- **생년월일** (만 24세 이상)
- **성별** (남성/여성/기타)
- **전화번호** (SMS 인증 완료된 번호)

**선택 정보:**

- **프로필 사진**
- **자기소개** (500자 이내)

### 2.2 자동 생성 정보

- **회원 ID** (시스템 고유 식별자)
- **가입일** (회원가입 완료일)
- **최근 로그인** (마지막 앱 접속일시)
- **회원 상태** (활성/정지/탈퇴)

### 2.3 활동 통계 정보(미정)

- **가입한 팀 개수**
- **참여한 경기 수**
- **평균 참석률**
- **받은 카드 현황** (옐로/레드/블랙카드)

---

## 🖼️ 3. 프로필 표시 및 관리

### 3.1 프로필 조회

**기본 정보 표시 (항상 공개):**

- 프로필 사진 (없으면 기본 이미지)
- 실명
- 나이 (생년월일 기준 계산)
- 가입한 팀 개수
- 참여한 경기 수

**상세 정보 (공개 설정에 따라 표시):**

- 성별 (공개 설정에 따라)
- 전화번호 (공개 설정에 따라)
- 자기소개 (공개 설정에 따라)

### 3.2 프로필 수정

**언제든지 변경 가능:**

- 프로필 사진 (5MB 이하, JPG/PNG)
- 자기소개 (500자 이내)
- 개인정보 공개 설정

**제한적 변경 (SMS 재인증 필요):**

- 전화번호 변경

**변경 불가능 (고객센터 문의 필요):**

- 실명
- 생년월일
- 성별

**시스템 관리 (변경 불가):**

- 회원 ID
- 가입일
- 활동 통계
- 소셜 로그인 방식

---

## 🔒 4. 개인정보 공개 설정

### 4.1 공개 범위 설정

**항상 공개 (설정 불가):**

- 실명 (팀 활동의 신뢰성을 위해 필수 공개)
- 프로필 사진
- 나이 (만 나이)

**공개 범위 설정 가능:**

- **전화번호**: 기본값 "팀 멤버만"
- **성별**: 기본값 "전체 공개"
- **자기소개**: 기본값 "전체 공개"

### 4.2 공개 범위 옵션

- **전체 공개**: 모든 Lockr 회원이 볼 수 있음
- **팀 멤버만**: 같은 팀에 속한 멤버들만 볼 수 있음
- **비공개**: 나만 볼 수 있음

### 4.3 공개 설정 적용 규칙

**팀 가입 시:**

- "비공개" 설정된 정보는 팀 가입 시에도 다른 멤버에게 공개되지 않음
- 예: 전화번호 "비공개" 시 긴급상황에도 다른 멤버가 연락 불가

**팀 내에서:**

- "팀 멤버만" 설정 시 같은 팀 멤버들에게만 표시
- 팀 탈퇴 시 해당 팀 멤버들은 더 이상 정보 열람 불가

## 🚪 5. 회원 탈퇴

### 5.1 탈퇴 프로세스

1. **탈퇴 의사 확인**

- 설정 메뉴에서 "회원 탈퇴" 선택
- 탈퇴 사유 선택 (선택사항)


2. **탈퇴 영향 안내**

- 모든 팀에서 자동 탈퇴됨
- 개인 데이터 영구 삭제됨
- 30일 후 복구 불가능
- 참석 예정인 일정들 자동 취소

3. **재확인 절차**

- SMS 인증을 통한 본인확인
- "탈퇴하기" 최종 확인


4. **탈퇴 완료**

- 즉시 로그아웃 처리
- 회원 비활성화
- 30일 후 데이터 완전 삭제

### 5.2 탈퇴 시 데이터 처리

**즉시 삭제:**

- 개인 프로필 정보
- 소셜 로그인 연동 정보
- 팀 멤버십 정보

**보관 후 삭제 (30일):**

- 회원 ID (재가입 방지)
- 기본적인 활동 로그 (통계 목적)

**영구 보관:**

- 법적 의무사항 (결제 내역 등)
- 익명화된 통계 데이터

## 🚨 6. 예외상황 처리

### 6.1 프로필 수정 에러

- 이미지 크기 초과: "5MB 이하의 이미지만 업로드 가능합니다"
- 자기소개 글자 수 초과: "자기소개는 500자 이내로 작성해주세요"
- 네트워크 오류: "네트워크 연결을 확인해주세요"
- 서버 오류: "일시적인 오류가 발생했습니다. 다시 시도해주세요"

### 6.2 회원 상태별 제한

**정지 회원:**

- 프로필 조회만 가능
- 수정/탈퇴 기능 제한
- "회원이 일시 정지되었습니다. 고객센터에 문의해주세요"

**탈퇴 진행 중 회원:**

- 30일 내 복구 가능
- 로그인 시 복구 의사 확인
- 복구 거부 시 즉시 완전 삭제

### 6.3 데이터 복구

- 실수로 삭제된 프로필 사진: 24시간 내 복구 가능
- 탈퇴한 회원: 30일 내 재가입으로 복구 가능
- 완전 삭제된 데이터: 복구 불가능
